## 属性

| 参数     | 类型   | 说明               |
| :------- | :----- | :----------------- |
| to       | number | 动画开始时数字大小 |
| from     | number | 数字大小           |
| duration | number | 动画持续时间(ms)   |

## Slot

| name   | 说明         |
| :----- | :----------- |
| prefix | 数字前缀内容 |
| suffix | 数字后缀内容 |
